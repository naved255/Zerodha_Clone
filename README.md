# Zerodha Clone Project

This repository contains a **Zerodha Clone** project including:

- **Frontend:** Zerodha UI clone
- **Dashboard:** Trading dashboard with holdings, positions, orders, and funds
- **Backend:** Node.js + Express API with MongoDB for storing users, holdings, and orders
